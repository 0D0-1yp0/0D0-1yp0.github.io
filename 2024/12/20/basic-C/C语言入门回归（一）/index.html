# 取位

说实话半年没摸真忘了是取位，还以为 a100+b10+c 这种操作来着

难度不大，就是师弟不知为何就是不懂，差点没把我气死矣

上 wp

#include<stdio.h>
int main()
{
  for (int i = 10; i <=100; i++)  if (i%10 == i/10) printf("%d ",i); 
  for (int i = 100; i <=1000; i++) if (i%10 == i/100) printf("%d ",i); 
  return 0;
}

如果不知 1000 的话就得用表达式表示范围了，以后再说
